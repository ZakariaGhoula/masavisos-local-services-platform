# ADR-002: Use WhatsApp as Primary Communication Channel

## Status

Accepted

## Context

The target market in Spain heavily uses WhatsApp for service coordination. Reducing friction is more important than forcing users into an app.

## Decision

Use WhatsApp as the primary client and professional communication channel.

## Consequences

Positive:
- Low user friction
- High adoption potential
- Faster response loops

Negative:
- Dependency on WhatsApp Business API
- Template and conversation pricing constraints
- Requires careful message compliance
